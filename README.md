# MangaK Extension

A [Paperback 0.8](https://paperback.moe) extension for [mangak.io](https://mangak.io).

## Install

Open the repo page and tap **Add to Paperback**:

**https://i-am-problems.github.io/manga-k-extension/**

Or paste that URL into Paperback under *Settings → External Sources → Add Source Repository*.

## Develop

```
npm install
npm run bundle   # writes ./bundles/
npm run serve    # serves bundles over LAN for sideloading
```
